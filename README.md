# Sushi Cart Application

A simple and functional shopping cart application for a sushi restaurant. Built to practice dynamic DOM manipulation and state management.

## Features

- Add to Cart: Add sushi rolls to the cart directly from the product list.
- Menage Quantity: Increase and decrease the number of items.
- Dynamic Calculation: Automatically updates the total price and delivery status based on the items in the cart.
- Responsive Design: Clean layout for better user experience.

- ## Technologies Used

- JavaScript(ES6
- HTML5 & CSS(Flexbox)
- LocalStorage
